28 ### Installation
29 
30 #### Requirements
31 * C++17 compatible compiler
32 
33  **MSVC is not supported currently due to cryptic errors. If you know how to fix them - please, send PR.**
34 
35 * _(Optional)_ Google Test for tests (included as submodule)
36 * _(Optional)_ Doxygen for API documentation generation
37 
38 ```
39 $ git clone https://github.com/foxcpp/wire
40 $ cd wire
41 $ mkdir build; cd build
42 $ cmake .. -DCMAKE_BUILD_TYPE=Release
43 $ cmake --build .
44 ```
45 
46 **Note 1** By default libwire will be built as a static library, if you want
47 a shared library then you should add `-DBUILD_SHARED_LIBS=ON` flag.
48 
49 **Note 2** It's recommended to enable LTO in your compiler to allow
50 cross-object inlining and stuff.

53 ### Usage
54 
55 #### Build system configuration
56 
57 If you use CMake and clone libwire as a submodule, you can just link it this way:
58 ```cmake
59 add_subdirectory(wire/)
60 target_link_libraries(your-exectuable libwire)
61 ```
62 It will use CMake magic to configure include path and other stuff.
63 
64 Alternatively, you can access system-installed version using `find_package`.
65 ```cmake
66 find_package(libwire REQUIRED)
67 target_link_libraries(your-executable libwire)
68 ```

71 #### In code
72 
73 Let's write a simple TCP server that echoes back `\n`-terminated strings.
74 
75 First, include TCP listener header and another one for `ipv4::any`:
76 ```cpp
77 #include <libwire/tcp/listener.hpp>
78 #include <libwire/address.hpp>
79 ```
80 
81 Then... It's very short so it doesn't needs any commentary:
82 ```cpp
83 using namespace libwire;
84 
85 tcp::listener listener({ipv4::any, 7});
86 
87 while (true) {
88  auto sock = listener.accept();
89  auto str = sock.read_until<std::string>('\n');
90  sock.write(str);
91 }
92 ```
